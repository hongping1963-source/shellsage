## 特别说明

✨ **零依赖安装**: VS Code TheFuck 是完全自包含的扩展，内置了所有必需组件：
- 独立的 Python 环境
- 预装的 TheFuck
- 所有必需的依赖

🚀 **开箱即用**: 安装后即可使用，无需任何额外配置。

## 获取帮助

如果你在使用过程中遇到问题：
1. 查看 [用户指南](./USER_GUIDE.md) 中的故障排除部分
2. 检查 [安装指南](./INSTALLATION.md) 确保正确安装
3. 提交 GitHub Issue 寻求帮助
